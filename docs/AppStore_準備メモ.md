# App Store 申請準備メモ

## アプリ情報

### アプリ名
叱咤激励

### サブタイトル（30文字以内）
偉人の名言が1日のスイッチを入れる

### カテゴリ
- プライマリ: ライフスタイル
- セカンダリ: 仕事効率化

### 説明文
```
惰性で生きるな。
偉人の言葉で、1日のスイッチを入れる。

「叱咤激励」は、ただの目覚ましじゃない。
ナポレオン、坂本龍馬、エジソン、宮本武蔵──
歴史に名を刻んだ偉人たちが、設定した時間にあなたを叩き起こす
新感覚のモチベーション・自己啓発アプリです。

■ こんな人のために作った
・朝起きてもすぐ動き出せない
・月曜が憂鬱で気合いが入らない
・自己啓発本を買っても3日で読まなくなる
・「今日も惰性で過ごしてしまった」と寝る前に後悔する

そんな自分を、偉人の言葉で叩き直す。
一発で目を覚ます、強制モチベーション装置。

■ 3つのモードで気分に合わせて使い分け
🌅 叱咤モード — 朝から喝を入れて目を覚ませ
🔥 追い込みモード — 油断した自分を追い込め
🎉 はっちゃけモード — 休日は全力で楽しめ

平日は叱咤モードで気合い注入、週半ばは追い込みモードでギアを上げ直し、
休日ははっちゃけモードでリフレッシュ。
あなたのライフスタイルに合わせた「気合い注入スケジュール」を組めます。

■ 偉人の名言が「声」で届く
ナポレオン「不可能という文字は愚か者の辞書にしか存在しない」
坂本龍馬「世の人は我を何とも言わば言え。我がなすことは我のみぞ知る」
エジソン「私は失敗したのではない。うまくいかない方法を1万通り見つけたのだ」
リンカーン「未来の最善の予測は、それを自分で作ることだ」
宮本武蔵「千日の稽古を鍛とし、万日の稽古を練とす」

これらの言葉が、設定した時間に音声で読み上げられます。
画面を見るだけじゃなく「耳」から染み込ませる、新感覚の自己啓発体験。

■ 自由なスケジュール設定
曜日ごとにモードと時間を自由に組み合わせ可能。
平日朝6時は叱咤モード、水曜昼は追い込みモード、金曜夜ははっちゃけモード──
あなただけのモチベーション・ルーティンを構築できます。

■ リアルタイムカウントダウン
次の叱咤激励までを秒単位で表示。
「あと何時間で偉人に叩き起こされるか」が常に見える緊張感が、自然と背筋を伸ばします。

■ 継続的にメッセージ追加
偉人の名言・叱咤激励メッセージは継続的にアップデートで追加。
飽きずに毎日新鮮な刺激を受けられます。

■ 買い切り型
一度購入すれば追加課金なし。
将来追加されるメッセージも自動的に利用可能です。

惰性で生きるか、偉人の言葉で動くか。
今日からあなたの1日を変えよう。
```
