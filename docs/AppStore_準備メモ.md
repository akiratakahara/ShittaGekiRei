# App Store 申請準備メモ

## アプリ情報

### アプリ名
叱咤激励

### サブタイトル（30文字以内）
偉人の名言で目覚める最強アラーム

### カテゴリ
- プライマリ: ライフスタイル
- セカンダリ: ヘルスケア/フィットネス

### 説明文
```
毎朝、偉人の名言があなたを叩き起こす。

「叱咤激励」は、曜日ごとにモードを設定できるモチベーション目覚ましアプリです。

■ 3つのモード
🌅 叱咤モード — 朝から喝を入れて目を覚ませ
🔥 追い込みモード — 油断した自分を追い込め
🎉 はっちゃけモード — 休日は全力で楽しめ

■ 偉人の名言で目覚める
ナポレオン、ゲーテ、坂本龍馬、リンカーン、エジソン、宮本武蔵...
歴史に名を刻んだ偉人たちの言葉が、あなたの朝を変える。

■ 音声読み上げアラーム
設定した時間に叱咤激励メッセージを音声で読み上げ。
テキストだけでなく「声」で叩き起こされる新感覚の目覚まし体験。

■ 自由なスケジュール設定
曜日ごとにモードと時間を自由に設定可能。
平日は叱咤モードで気合いを入れ、休日ははっちゃけモードでリラックス。

■ リアルタイムカウントダウン
次の叱咤激励までのカウントダウンを秒単位で表示。
緊張感を持って毎日を過ごせ。

■ 買い切り型
一度購入すれば追加課金なし。将来追加されるメッセージも自動的に利用可能。
```
